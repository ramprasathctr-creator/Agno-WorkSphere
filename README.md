# Agno WorkSphere

A modern, comprehensive project management platform built with FastAPI and React.

## Project Structure

This repository is divided into two main components:

- **[Frontend (FE)](file:///d:/PM/fixing%20V3/FE)**: A React-based web application with Kanban boards, role-based dashboards, and real-time notifications.
- **[Backend (pms-bug-fixing-backend)](file:///d:/PM/fixing%20V3/pms-bug-fixing-backend)**: A FastAPI-powered REST API managing project data, user roles (RBAC), and AI-driven insights.

## Key Features

- **Multi-Tenant Organizations**: Seamless switching between different organizations with unique settings.
- **Role-Based Access Control (RBAC)**: Specific views and permissions for Owners, Admins, Members, and Viewers.
- **Kanban Task Management**: Dynamic task tracking with drag-and-drop and assignment logic.
- **Real-time Notifications**: In-app alerts and email notifications for project updates.
- **AI-Driven Project Summaries**: Automated project status reporting powered by OpenAI.
